Compiler Design Lab

Overview

This repository contains implementations and experiments related to the Compiler Design subject. It demonstrates the fundamental phases of a compiler, including lexical analysis, syntax analysis, semantic analysis, and intermediate code generation.

The project is designed to provide practical understanding of how high-level programming languages are processed and translated into machine-level instructions.

Objectives
To understand the working of a compiler
To implement different phases of compilation
To design lexical and syntax analyzers
To generate intermediate code representation
To explore error detection and handling techniques
Modules Implemented
1. Lexical Analysis
Tokenization of input source code
Identification of keywords, identifiers, operators, constants
Removal of whitespace and comments
2. Syntax Analysis (Parser)
Implementation of parsing techniques (LL / Recursive Descent)
Construction of parse trees
Syntax error detection
3. Semantic Analysis
Type checking
Scope resolution
Symbol table management
4. Intermediate Code Generation
Generation of three-address code (TAC)
Expression evaluation
Temporary variable handling
5. Code Optimization (Optional)
Constant folding
Dead code elimination
Strength reduction
